# Python for Data Science 🐍📊

A hands-on journey from core Python toward Data Science, built as a
collection of self-contained Jupyter notebooks, one per topic.

## Topics

- Python fundamentals (variables, data types, operators)
- Collections, control flow, functions, OOP
- NumPy, Pandas, and visualization (more coming)
